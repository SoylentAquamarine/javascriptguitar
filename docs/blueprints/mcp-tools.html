MCP Tools — Javascript Guitar

Blueprint · Last updated 2026-05-02

Not Applicable

Javascript Guitar does not expose or consume any Model Context Protocol tools. It is a static client-side site with no AI integration, no backend, and no programmatic interfaces of any kind.

MCP is a protocol for AI agents to invoke tools in external systems. Javascript Guitar has no external system for an agent to call into.

MCP in This Ecosystem

MCP tooling exists in this project family via The Brain — the AI orchestration engine used across all projects in this portfolio. The Brain exposes a single delegate tool that routes tasks to free AI providers.

See the The Brain — MCP Tools blueprint for full documentation on registration, input schema, and response format.
