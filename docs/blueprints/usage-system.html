Usage System — Javascript Guitar

Blueprint · Last updated 2026-05-02

No Usage Tracking

Javascript Guitar does not collect, store, or transmit any usage data. There are no analytics scripts, no cookies, no localStorage writes, and no server-side logging of visitor behavior.

The site is a pure client-side experience. No data leaves the browser, and no data is retained between sessions.

Why This Does Not Apply

Usage tracking systems exist to optimize routing, understand user behavior, or monitor costs. None of those goals apply here:

  • There is no routing to optimize — the site is static.
  • There are no API costs to monitor.
  • There are no user accounts or sessions.
  • Privacy is preserved by design — no tracking infrastructure exists.

GitHub Pages Traffic

GitHub provides basic traffic analytics (views, unique visitors, referring sites) in the repository's Insights tab. This is the only usage data available for the project and is read-only — it cannot be accessed programmatically from within the site itself.
