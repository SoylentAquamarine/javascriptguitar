# javascriptguitar

https://soylentaquamarine.github.io/javascriptguitar/

This is a javascript project from when I was first learning HTML, I re-worked it with Claude and made it a lot better.

## About

An interactive guitar fretboard visualizer built in vanilla JavaScript. Displays chord shapes, scales, barre chords, and fingering diagrams directly in the browser — no frameworks, no dependencies.
