Leave Notes on Neck

Click a note to show that note on the neck. Change the option to leave the note on the neck, and if you click C, E, and G, then you can see your C major triad in all positions.

This uses the Javascript image preload and flip functionality that was new 30 years ago when I wrote this as an exercise. There are a few bugs and the whole thing is incomplete, I will be tweaking things. But, it is an example of what I used in 1996 and 1997 to learn HTML and Javascript.

I made all of the little graphics myself using Paintshop Pro. Took me a while to get it to look right, with the length, width, number of pixels, etc. You can view source on this or any of the jsguitar pages and see how I did it ... I had no idea what I was doing, so I am sure it is not the best way. But, it currently works on Chrome, Edge and Firefox browsers. It used to take up the whole screen back in the 90's, I think I had my screen set at 800x600. Funny.
